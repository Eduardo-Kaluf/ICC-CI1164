# Exercicio 4

Calculando a integral da função styblinskiTang para 2 dimensões utilizando o método de monte carlo e o método dos retângulos

Compile it:

```yaml
gcc carloAndRet.c utils.c -lm
```

## Observações

Inicialmente este programa ser genérico em questão das dimensões a serem calculadas, por isso foi implementado com vetores. Depois a mudança foi feita para ser apenas em 2 variáveis, sendo assim, existem muitas maneiras melhores de otimizar este programa considerando uma versão com dimensões fixas
